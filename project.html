Getting Started with City Timezone Finder

Welcome to the City Timezone Finder! Here's how you can quickly find the timezone information for cities around the globe:

  1. Enter City Name: Start by typing the name of the city you're interested in. Our app supports cities worldwide.
  2. View Timezones for Major Cities: Only major cities will have timezone information available. Examples of cities you can search for include New York, Tokyo, Panama City, London.
  3. Handling Incorrect City Names: Make sure to spell the city names correctly. If a city name is misspelled, like "New Torks," you'll see an error message: "No data found for a city called New Torks."
  4. City Limits: You can add up to 3 cities. If you try to add more, you'll need to hit the Reset button to clear your selections. An error message will appear: "You have reached the limit of 3 cities."
  5. Service Availability: Our app relies on a Public API to fetch timezone information. If the service is temporarily unavailable, you'll receive a notification stating that the service is not available.

Enjoy discovering timezones across the world with our easy-to-use app. Happy searching!
